Decorative Icon

Decorative icons are used for visual or branding reinforcement. Removing a decorative icon from a page or component should not alter a user's ability to use or understand it.

Theming

Accessibility

Decorative icons have no semantic meaning and should not be used with a <button> element to prevent it from becoming a focusable, interactive element. The aria-hidden=true attribute is added by default to reinforce that they are hidden from assistive technologies.
